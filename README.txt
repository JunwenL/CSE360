The program includes javafx package, so we have to install JDK8(javafx package included) or download the openJDK of javafx as an extension to the JDK other than JDK8, such as java11, java 13, or java17. 

Here is the way of running the code in VS Code properly:

In Visual Studio Code, first click on File - Open Folder... - CHOOSE THE FOLDER OF "server".
Once the "server" folder is opened in VS Code, click on src/GroupProject - Main.java, and run "Main.java"
After that, start a new window of VS Code, and click on File - Open Folder... - CHOOSE THE FOLDER OF "client".
In the "client" folder, find "Main.java" and click on "run java" button which you can find on the up right corner.
